Directions to Navigate Alpha Team's app while watching the video

1. Use the "live app" link to go to the most recent version of the app
2. Sign up with an email and password
3. Log in with email and password
4. Select the Employees tab. Click on the Plus button on the bottom right to add an employee. The new employee and his/her information will be displayed.
5. Select the Form Editor tab to make a unique form. The save feature is coming soon, only for our Premium members.
6. Select the Schedule Appointment tab. Enter Appointment information and click "Create Appointment" to schedule an appointment. You will receive a verification code.
7. Once the appointment is created, if you entered a valid phone number, you will get a phone call from the SMS bot. Input the verification code.
8. The bot will message you and you will be able to interact with the bot.
